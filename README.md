> **CultureEvents** — это веб-приложение для поиска и бронирования билетов на концерты, театры, выставки и фестивали. Проект разработан на Django с использованием современных технологий.

## 🚀 Возможности

- ✅ **Аутентификация пользователей** — регистрация, вход, выход
- ✅ **CRUD мероприятия** — создание, чтение, обновление, удаление событий
- ✅ **Категории и теги** — удобная навигация по мероприятиям
- ✅ **Пагинация** — разбивка списка на страницы
- ✅ **Поиск и фильтрация** — быстрый поиск по названию, категории, тегам
- ✅ **ЧПУ-ссылки (slug)** — человекопонятные URL
- ✅ **Кастомный менеджер** — выборка только опубликованных мероприятий
- ✅ **Админ-панель** — управление контентом с превью изображений
- ✅ **Яндекс.Карты** — отображение офиса на карте
- ✅ **AI-помощник** — чат с искусственным интеллектом (OpenRouter API)
- ✅ **Адаптивный дизайн** — работает на всех устройствах
